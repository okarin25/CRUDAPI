API Documentation

Product API

Endpoints

POST /api/product/createProduct: Create a new product
Request Body:
name: string (required)
description: string (required)
price: number (required)
category: string (required)
stock: number (required)
Response:
201 Created: Product created successfully
500 Internal Server Error: Error creating product
GET /api/product/getAllProducts: Get all products
Response:
200 OK: Products fetched successfully
404 Not Found: No products found
500 Internal Server Error: Error fetching products
GET /api/product/getProductById/:id: Get a product by ID
Path Parameters:
id: string (required)
Response:
200 OK: Product fetched successfully
404 Not Found: Product not found
500 Internal Server Error: Error fetching product
DELETE /api/product/deleteProductById/:id: Delete a product by ID
Path Parameters:
id: string (required)
Response:
200 OK: Product deleted successfully
404 Not Found: Product not found
500 Internal Server Error: Error deleting product
